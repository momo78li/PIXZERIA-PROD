# PIXZERIA - Webdesign so einfach wie Pizza bestellen

## Über das Projekt

PIXZERIA ist eine moderne Webdesign-Agentur, die sich auf einfache, transparente und schnelle Website-Erstellung spezialisiert hat. Unser Motto: "Webdesign so einfach wie Pizza bestellen".

## Geschäftsmodell

- **Digital First**: Alle Prozesse laufen online ab
- **Feste Pakete**: Keine versteckten Kosten
- **Einmalige SEO**: Keine dauerhaften SEO-Verträge
- **Optionales Hosting**: Kunden können zwischen eigenem Hosting und PIXZERIA Hosting wählen

## Technologie Stack

- **Frontend**: React 18 mit TypeScript
- **Backend**: Node.js mit Express
- **Database**: PostgreSQL mit Drizzle ORM
- **Styling**: Tailwind CSS + Shadcn/ui
- **Deployment**: Vercel + GitHub Actions

## Entwicklung

```bash
# Entwicklungsserver starten
npm run dev

# Projekt bauen
npm run build

# Produktionsserver starten
npm run start

# Datenbank Schema pushen
npm run db:push
```
